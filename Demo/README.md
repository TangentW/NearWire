# Demo

This directory owns the maintained iOS integration application for NearWire.

The Demo will exercise Swift Package Manager and CocoaPods integration paths against one shared application implementation. It will cover pairing-code discovery, bidirectional Codable events, optional NearWireUI, Viewer control handling, and NearWirePerformance.

The manually maintained `NearWireDemo.xcodeproj` will be created in the dedicated Demo implementation change.
